# dipta-seu-theme

Foobar is a Python library for dealing with word pluralization.

## Requisites

Gulp 4.0.2

Node 12.18.4

Liferay-theme-tasks ^10.0.2

```bash
npm install -g yo gulp
npm install -g generator-liferay-theme@10.x.x
```

## Usage

```bash
yo liferay-theme
gulp build
```
